使用Fastcomet五分鐘架設WordPress教學

  1. 首頁
  2. 網路資源精選
  3. 使用Fastcomet五分鐘架設WordPress教學

架設網站或是網路商店需要租用主機和網域，但現在已經有部分的廠商同時提供租用虛擬主機和提供免費網域的服務，但我們在挑選主機商初期會不曉得該如何從何下手，這篇文章就是要教同學如何挑選主機廠商，並且如何在虛擬主機上安裝Wordpress。

挑選虛擬主機商的參考指標有哪些？

挑選主機商有以下幾個重要的參考指標：

  • 客戶服務品質與技術支援
  • 主機穩定性
  • 流量限制
  • 網站空間
  • 付款方式

其中我認為「客戶服務品質與技術支援」最為重要，因為我們在開始架設網站初期難免會遇到操作上的疑問，或是在網站在營運過程中需要客服的技術支援，優質的「客戶服務品質與技術支援」尤其重要

在我過去的架站和架設電商網站的經驗，前前後後使用了十多個主機商的經驗中發現，流量、空間可以在付款時自行選擇適合的方案，但好的「客戶服務品質與技術支援」是無法用錢衡量的，他只能透過使用者經驗分享，還有國外第三方評分網站的評價才能知道主機商的客戶服務品質，在我使用多個主機商中Fastcomet是最好的一間，你可以在下圖看到國外評比的分數高達9.7分，尤其是在客戶服務的品質2017年拿下第一名的勳章，爾後也一直維持著高水準的服務品質，直到現在我仍有多個網站架設在Fastcomet，當我需要協助時他們總是會幫我搞定一切。

 

高水準的主機穩定性

在挑選主機商的時候，我們非常重視主機的穩定性，如果主機不穩定停止服務，造成的後果除了是你的客戶無法連上網站的金錢損失，也影響了搜尋引擎對你網站的信心，網站排名可能因此往後，所以主機穩定性至關重要。

通常參考依據為正常運作時間(Uptime)，國外網站Hosting Facts追蹤各個主機商的正常運作時間(Uptime)，並客觀的將資料紀錄統計下來，我們可以看到Fastcomet的正常運作時間(Uptime)都幾乎是100%，由客觀的數據可以看到Fastcomet是一個能夠提供主機穩定的主機商。
(最新的數據你可以參考Fastcomet近幾個月正常運作時間(Uptime) )

主機使用速度較快的固態硬碟(SSD)，網路流量無上限

確認客戶服務與主機穩定性後，我們就可以挑選適合我們的網站空間與流量限制，Fastcomet目前的三個方案都提供網路流量無上限的服務，你不用擔心你的網站快速成長後流量增加，你是否該更換方案或是轉換主機商，因為網路流量無上限的服務讓你可以無後顧之憂，就是一個吃到飽的概念。

網路空間有15 GB, 25 GB, 35 GB的選擇，主機使用速度較快的固態硬碟(SSD)。若你是使用Wordpress這方面的套件架設網站，你大概只需要1~2 GB左右的硬碟空間就很足夠，當容量高達35 GB的時候，相當於儲存35萬張左右的圖片。而且固態硬碟(SSD)的速度較快，相較於傳統硬碟(HDD)速度會快上4~5倍左右，能夠提供更好的服務品質。但也因為固態硬碟(SSD)的成本比較高，所以這是一般虛擬主機商比較少提供的服務，然而Fastcomet的價格方案卻意外的划算，稍後下面的文章將介紹優惠方案內容。

免費的網域名稱(Domain)

架設網站或商城最重要的兩件事，除了主機以外就是網域名稱，購買網域的廠商也很多，但你要知道的是網域購買初期便宜，不代表來年你續約的價格也如同第一年般便宜，大部分續約的價格都會提高，而你又捨不得丟棄你長期經營頗有成效的網域名稱，只能咬著牙繼續買下去，Fastcomet主機商提供免費的網域名稱，只要你租用他的主機服務，他就會提供你免費的網域名稱，而且是Forver，只要你是他們的客戶你就能持續享用免費的服務，如果你原本就已經有在其他地方購買的網址也可以搬移過去。

多種軟體免費安裝

Fastcomet提供多種免費的軟體套件供我們使用，這些軟體都是免費的，安裝過程也都非常簡便，這篇文章將會示範如何使用他們的服務安裝Wordpress，你只要依步驟執行即可，另外，你可以看到在左方的選單有提供許多解決方案，像是公司用的ERP系統，E-Commerce相關的軟體套件都可以直接安裝使用。

 

免費搬移主機資料

若你在其他主機商已經有架設網站，想要搬移主機資料到Fastcomet上，他們提供了免費搬移主機的服務，有些主機商也有這類的服務，但大都是只提供教學文件請你自己搬，或是用使用第三方軟體執行，Fastcomet則有一個專門搬移主機資料的專業團隊，他們會幫你把搬移主機這件事搞定，你不需要執行任何指令，或上網查詢該如何搬移資料庫的資料和圖檔，他們大約在一個小時內就可以完成搬移主機的流程，如果你在其中需要其他服務，他們也會客製化幫你處理，這是與一般主機商最大的不同之處。

速度快又專業的客服與技術支援

Fastcomet的客服是透過他們的系統發送問題，這個系統類似E-mail的方式管理你所提問的問題，由於Fastcomet的客服是由一個專業團隊負責，每個問題都會有一個專屬的ID，你可以查閱過去的問題，也可以確認每一個問題目前的狀態，這個好處是在於可以監督每位客戶所提問的問題都能被滿意的回覆，並且能夠在最短的時間獲得你需要的答案，你會看到他們如何有效控管他們的服務品質，這公開透明的方式也讓我們充分理解Fastcomet為什麼能夠提供最好服務品質的原因，以下是我截圖是我使用的系統後台畫面，每次的問題都很有耐心的回覆，我也都得到滿意的答案。

 

我使用其中一個問題分享我自身的經驗，我詢問了一個設定的問題，提問的時間是在7:03分，而我取得回覆的時間是在7:06，中間只有短短的3分鐘，如果換作是我打字可能就不只3分鐘了 (>_<“)你可以看到他回覆的右下角是我對他回覆的評分，評分標準從1到5顆星，所以他們要求的不只是速度也要求客服人員回覆內容的品質。由於他的回覆解決了我的問題，所以我給他5顆星，這樣他在內部績效評比就會高，優秀的客服人員獲得獎勵，我們得到了更好的服務，如此正向循環，就能看出Fastcomet的客服被人評價為高水準是有他們用心之處。

 

具有競爭力的價格方案

Fastcomet有三種價格方案，分別是最實惠的每個月2.95美元方案，中階方案的每個月5.95美元，還有加快速度優化系統的SpeeUp每個月也只要9.95美元，相當於每個月不到300元台幣，就能夠享有最高等級的服務，無限制的網路流量、35 GB的固態硬碟(SSD)空間還有終身免費的網域。

 

有些功能是沒有在這張表格中展示的特色，像是營運網站時所需要的各項服務與功能，相較於其他主機商，Fastcomet免費提供這些實用的服務，否則你挑選了一間便宜的主機商但未來仍然要付出其他額外的成本和設定的技術門檻，例如：每天自動備份功能、SSL憑證、即時聊天服務、全世界主機機房選擇(台灣可以選擇就近的亞洲機房)、防火牆…等，這些服務是我們在挑選主機商時要考慮到的細節。

如果你還不確定？時間長達45天，又確定有效的「保證退款」

看完上面的介紹你心裏還是不確定是否要購買，這是很正常的，因為這不像是大賣場的商品可以拿起來摸一摸把玩一下，東西還是要試用過才知道好不好，你可以先按照下面教學步驟開始架設網站，之後你還有長達45天的退款時間，你可以在上面價格方案中的圖示中看到標題第二行有保證45天退款的說明，45天內如果你想要退款，Fastcomet不囉唆直接退款給你，還會謝謝你給他們機會，未來希望能再為你服務。

網站名稱：Fastcomet

網站連結：https://go.codegym.tech/fastcomet

步驟1

首先請點選上面網址進入到Fastcomet官網，映入眼簾的首頁是一張全世界的地圖，它顯示了Fastcomet所有機房的位置，稍後的設定中你可以選擇亞洲的兩個機房中的一個，請先點選「START NOW」的按鈕。

 

選擇一個適合您的方案，如前面所述分別有每個月2.95美金、5.95美金和9.95美金三種方案，如果你只是剛起步而且只有一個網站可以選擇每個月2.95美金的方案，但我會建議你選擇9.95美金「SpeedUp」方案，因為他的有優化過，速度更快，功能更多、網路空間更大，而且換算下來平均一個月不到300台幣。

 

步驟2

剛才在前面有介紹過Fastcomet有提供免費的網域，我們將在這一步驟選擇你的網域名稱，如果你已經有網域了請選擇右邊的選項「I already Have a domain」。

 

在選擇網址的過程中，如果你要選擇的網址名稱已經被購買走了，系統會在下面推薦你其他的網域名稱，都是依據你原先輸入的名稱做推薦，最後選擇你要的網域後請選擇「Use This Domain」按鈕就完成了網域申請的設定。

 

步驟3

請填入個人資料與聯絡方式：

  • First Name：請填入「名字」，Last Name請填入「姓氏」
  • Company Name：公司名稱是選填的，如果你是個人經營的網站可以不用填寫
  • Address：地址請寫英文，你可以使用郵局提供的中文地址轉英文地址的線上服務填入這一欄
  • E-mail填入後，在Phone Number 電話號碼這邊請記得加上國碼+886，電話開頭的0可以不用填寫，例如：+886921XXXXXX。
  • Master Password：請輸入之後你要進入Fastcomet 後台管理系統的密碼，這裡設定好之後會使用你的E-mail當作帳號和此密碼作為登入驗證資料。
  • Zip Code 請依據你的住址，參考郵局3+2郵遞區號查詢服務。
  • City：就選擇你住的城市，例如：Taipei。
  • State/Region：請輸入Taiwan，參考建議來源。

右邊是結帳總金額，其中有一個項目是Domain privacy，他可以保護你的網域個人資料被其他人看到，他是選擇性購買，但我強烈建議你要購買此項目，因為你不會希望你的名字、住址被大家查到，一年2.95美金相當划算，可別因小失大喔。
(有關WHOIS的說明可以參考wiki解釋)

另外你也可以看到你的網域的收費是Free。

 

步驟4

請選擇你要購買方案的時間長度，請留意要購買3年才有每個月9.95美金的金額，如果你覺得太長了可以先使用12個月的長度，但我會建議你直接選擇36個月的時間，因為別忘了你還有45天退款保證，這1個半月的時間夠你體驗一個優質的主機服務，45天候你想繼續使用Fastcomet的服務時，你此時選擇36個月的時間會是明智之舉。

 

步驟5

剛才步驟3有提到額外的費用每年2.95美金，他一開始已經幫你加入到購物車，這是用來保護你的網域個資不被其他人瀏覽，至於其他像是提升SEO的額外工具，你可以不用購買，我會撰寫其他增加SEO的相關教學文章幫助你的網站優化SEO。

 

步驟6

在Locations 下拉選單是選擇你預設主機的位置，在這邊我選擇「新加坡」，這也是Fastcomet給我的建議，實際上速度測試還蠻不錯，可以做為第一次設定機房的地點，若是想要調整的話未來也還可以再變更設定。選擇付款方式，Fastcomet有提供PayPal的付款方式，你也可以選擇信用卡付款。最後點選「Create Account」就完成申請手續開始使用主機的服務了。

 

完成申請後，你要進入後台開始安裝Wordpress，你會收到確認E-mail，裡面會有後台連結網址，或是使用首頁右上方的「Client Login」登入，帳號/密碼請輸入E-mail還有你在步驟3中設定的密碼。

 

步驟7

登入後台畫面你會看到你的服務狀態是一個綠色的ACTIVE，右邊則是功能選單，請點選「cP」圖示的按鈕，「cP」是代表cPanel的意思，進入後就能夠管理主機上的所有服務和設定。

 

以下就是cPanel的操作畫面，我們接著要開始安裝Wordpress，請下滑至SOFTACLOOUS APPS INSTALLER區塊，我們要安裝的軟體套件都在這個功能選項中。

 

步驟8

左邊的功能選單將所有軟體依功能分類，首頁第一個就是Wordpress，請直接點選即可，未來如果有想要在主機上安裝其他套件提供服務可以在這功能選單尋找。

 

步驟9

以下我們要對我們的網站做一些初始化的設定，這些設定大致上就是網站名稱和密碼等簡單的資料，你在這裡輸入完之後也都可以在Wordpress的後台調整，所以如果還沒決定好網站名稱和介紹也可以之後再更新。

  • Choose Protocol : 請選擇你的網站使用http或是加密的https，如果你還不是很清楚https或是相關設定可以先選擇http即可，未來設定好SSL憑證再調整為https。
  • Choose Domain : 這裡是下拉選單，請選擇你的網域名稱，如果你有多個網域，都會出現在下拉選單當中。
  • in Directory : 你可以選擇你的網址後面是否要加入路徑，例如你的網域下面想要開一個部落格專屬網站，所以你可以設定/blog，那麼你這個網站的網址就會是http://mydomain.com/blog，如果你不想設定其他路徑，在這邊空白就好囉。
  • Site Name : 網站的名稱。
  • Site Description : 請輸入你對網站的敘述，簡短清楚的介紹，這是網友對你的網站第一印象。
  • Enable Multisite : 請不要勾選。

     

  • Admin Username : 請輸入你在後台管理員帳號名稱。
  • Admin Password : 請輸入你在後台管理員帳號密碼。
  • Admin Email : 請輸入你在後台管理員電子郵件。
  • Select Language : 請選擇你要使用的語言。
  • Limit Login Attempts : 有些網路的機器人會嘗試隨機測你的登入帳密，你可以選擇安裝這個外掛程式，避免這些網路機器人亂測你的登入帳密。
  • Select Theme : 最後你可以選擇一個佈景主題，但你可以不選擇，之後再選擇要付費的佈景主題或是官方提供免費的佈景主題。

最後請點選最下面藍色的按鈕install開始安裝，不用一分鐘的時間就可以完成。

 

步驟10

安裝完成後你就會看到下面的畫面，他會有兩個網址，第一個就是你自己網域的網址，另一個框框標示起來的就是後台的網址，其實就是在你的網域後面加上/wp-admin/的路徑就可以了。

 

點選你的網址後你就可以看到你剛完成的網站首頁囉！
